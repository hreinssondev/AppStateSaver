# AppStateSaver
App Snapshot/State Saving AND Restoring in the same app made EASY! No skills in the Terminal needed nor patience for a Time machine backup. 
******************************************

Example usage: How to setup on a new mac

    1) Save snapshots of the apps you want, they appear in the restore tab list

    2) Save all snapshots which makes a folder (save somewhere safe)

    3) On the new mac download all apps beforehand

    4) Instsall AppStateKeeper on new mac and open that saved folder. Apps saved should appear in restore tab list

    5) Click on Restore to this mac.
   
    Tada! No command line memorisation or lenghty time machine backup needed.
